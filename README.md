# Sistema Tally Roland V-1HD / Datavideo ITC-100

Sistema embedded per leggere lo stato Program e Preview della regia Roland V-1HD tramite MIDI e attivare segnali Tally compatibili con Datavideo ITC-100.

## Obiettivo

Il progetto nasce per creare un sistema Tally economico e personalizzabile per piccole regie video usando Raspberry Pi o ESP32 e moduli relè.

## Funzionalità previste

- Lettura dello stato camere da Roland V-1HD
- Gestione segnali Program e Preview
- Attivazione relè per Datavideo ITC-100
- Supporto fino a 4 camere (espandibili)
- Possibile espansione con interfaccia web o log eventi

## Hardware utilizzato

- Roland V-1HD
- Datavideo ITC-100
- Raspberry Pi / ESP32
- Modulo relè
- Cavi di collegamento
- Cavi di collegamento MIDI
